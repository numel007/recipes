# Copycat In-N-Out Spread

## Ingredients

| Name          | Quantity       | Configuration |
| ------------- | -------------- | :-----------: |
| Relish        | 4 tsp          |       -       |
| Mayonnaise    | 5 tbsp + 1 tsp |       -       |
| Ketchup       | 4 tbsp         |       -       |
| White vinegar | 1 tsp          |       -       |
| Sugar         | 1 tsp          |       -       |

_Yield: ~1.5 cups_

## Method

1. Combine all ingredients well.
