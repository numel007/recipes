# **Mozzarella Stuffed Meatballs**

## **Ingredients**
| Name | Quantity | Configuration |
| ---- | -------- | :-----------: |
| Ground beef | 1 lb | - |
| Bread crumbs | 1/2 cup | - |
| Milk | 1/4 cup | - |
| Egg | 1 whole | - |
| Mozzarella sticks | 4 pieces | 1/2 inch pieces |
| Worcestershire | 1/2 tsp | - |
| Salt | 1/2 tsp | - |
| Black pepper | 1/4 tsp | - |
| Thyme | 1/2 tsp | - |
| Oregano | 1/2 tsp | - |
| Garlic powder | 1/2 tsp | - |
<br>

*Yield: ~20 meatballs*
<br><br>

## **Method**
