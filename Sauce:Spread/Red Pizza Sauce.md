# Red Pizza Sauce

## Ingredients

| Name                  | Quantity | Configuration |
| --------------------- | -------- | :-----------: |
| Canned diced tomatoes | 14.5 oz  |    Drained    |
| Garlic                | 4 cloves |       -       |
| Olive oil             | 2 tbsp   |       -       |
| Sugar                 | 1 tsp    |       -       |
| Balsamic vinegar      | 1.5 tsp  |       -       |
| Kosher salt           | 0.75 tsp |       -       |
| Pepper                | 0.5 tsp  |       -       |

_Yield: ~1 cup_

## Method

1. Blend together all ingredients.
1. Use immediately.
